# Trader Kachi Website

A modern, responsive website built for **Trader Kachi**, a Nigerian trading and forex brand.  
The platform is designed to showcase services, build trust, and attract new clients through a clean, professional online presence.

---

## 🚀 Features

- Modern landing page design
- Responsive layout (mobile + desktop friendly)
- Fast performance using Next.js
- Clean UI for marketing and brand presentation
- Optimized for SEO and conversions

---

## 🛠️ Technologies Used

- Next.js (React Framework)
- TypeScript
- Tailwind CSS
- Vercel (Deployment)
- Git & GitHub (Version Control)
